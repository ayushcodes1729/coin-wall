Coin-wall is a web based wallet to generate and manage Solana and Ethereum wallets. It is built using Next.js, Tailwind CSS, and TypeScript. The project is designed to be simple, intuitive, and secure, allowing users to easily create and manage their cryptocurrency wallets.

## Features
- Generate Solana and Ethereum wallets
- View wallet balances and transaction history
- Send and receive cryptocurrency
- Securely store wallet information using local storage
- Responsive design for mobile and desktop devices

## Installation
To run the project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/ayushcodes1729/coin-wall.git
    ```
2. Navigate to the project directory:
   ```bash
   cd coin-wall
   ```
3. Install the dependencies:
   ```bash
   pnpm install
    ```
4. Start the development server:
   ```bash
   pnpm run dev
    ```
5. Open your browser and navigate to `http://localhost:3000` to view the application.
